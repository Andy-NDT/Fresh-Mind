# Fresh Mind
<img width="2123" height="1389" alt="FM hero" src="https://github.com/user-attachments/assets/d81152e3-bd54-4ec8-a8cd-9671fc4f9fa1" />

Дневник для тех, кто хочет видеть себя со стороны.

**Локально. Без облака. Без ИИ внутри.**

Mind не думает за тебя. Не интерпретирует. Не выдаёт советы «попробуй X».
Только отражает, помнит и хранит. А когда захочешь разбора — одной кнопкой
готовит структурированный отчёт для разбора в твоём ИИ-сервисе.

## Скачать

[**Последний релиз →**](https://github.com/Andy-NDT/Fresh-Mind/releases/latest) (текущая версия — **v0.3.0**)

## Что умеет

### 🧭 Колесо 20 сфер жизни
<img width="2007" height="1207" alt="COMPARE" src="https://github.com/user-attachments/assets/a13ef17f-fbac-49f5-9f71-0c04fa4012fc" />

Интерактивный radar-chart: 20 сфер по 4 группам (Здоровье / Развитие / Труд /
Общество). Шкала 0–10 на каждый день. Сферы и группы настраиваются: можно
переименовать, поменять цвет, добавить, удалить, перетащить порядок.

- Левый клик по сфере → шкала оценки
- Перемотка дат под колесом → срез на любой день
- Сравнение двух дат → две паутинки одновременно (как Strategium Wheel)
- Тренды по каждой сфере: 30 / 90 / 365 дней
- График общей средней с эмодзи-маркерами записей по времени

### 📓 Дневник
<img width="766" height="571" alt="NOTE" src="https://github.com/user-attachments/assets/bd876ee3-fabe-4c69-89bc-3c98a019b4be" />

TipTap-редактор: форматирование, заголовки, списки, цвета, выделение.
Эмодзи-настроение прикрепляется к записи. Теги через `#` с автокомплитом.
Прикрепление картинок. Связь записей со сферами и автоматическая запись
оценок.

- Полнотекстовый поиск (SQLite FTS5)
- Pinned-памятки сверху ленты — для своих правил и мантр
- Фильтры по тегу / сфере / диапазону дат / эмодзи / наличию вложений
- Корзина с восстановлением soft-deleted записей
- «В этот день» — записи из этой же даты в прошлые годы

### 📤 Шеринг — колесо, сравнение, динамика (v0.3.0)
<img width="2160" height="2160" alt="Sharing" src="https://github.com/user-attachments/assets/365aea68-5d1e-43b8-9710-fb6351e12df5" />

Кнопка-скачать рядом с колесом за день, в режиме сравнения двух дат и в
блоке динамики. Открывает модал с выбором:

- **Размер:** 1080×1080 (квадрат), 1920×1080 (широкий), 1200×630 (горизонтальная карточка)
- **Фон:** прозрачный, белый, тёмно-фиолетовый
- **Подпись по центру внизу** — опционально, до 40 символов

В углу картинки — маленький фирменный маркер **Fresh Mind**. Файл сохраняется
через нативный системный диалог Windows. Никуда не загружается, никаких
внешних запросов.

### ✨ Экспорт для ИИ-аналитики (v0.2.0)
<img width="1353" height="806" alt="ИИ анализ" src="https://github.com/user-attachments/assets/e8f7c1b6-96eb-4e03-981e-5cc315231a2b" />

**Главная фича релиза 0.2.0.** Окно в Настройки → Бэкап → «Экспорт для
ИИ-аналитики». Mind собирает твои записи и оценки сфер в структурированный
отчёт `.txt` с готовой инструкцией для разбора. Загружаешь файл в свой
ИИ-сервис — получаешь взгляд со стороны на свои паттерны и динамику.

**6 готовых типов разбора:**

- 🎯 Глубокий разбор периода — 7-секционная ретроспектива
- 🔍 Анализ одной сферы — что её поднимает / опускает по данным
- 💡 Помощь с решением — твой вопрос + опираясь на твои данные
- 🔦 Поиск слепых пятен — что не замечаешь сам
- ⚖️ Сравнение двух периодов — было / стало
- 💬 Свой промпт — любая задача, структуру разбора задаёшь сам

**В отчёте:**

- Динамика всех 20 сфер с трендом по третям периода
- Топ-10 ярких записей (выделены по pinned + настроению + сферам + длине)
- Паттерны: время записей, топ-теги, распределение настроений
- Корреляции Pearson между сферами — показываются только при ≥60 днях с
  оценками (гейтинг от ложных сигналов на малых выборках)
- Полные записи в хронологическом порядке

**Принципиально:** ни одного упоминания конкретных ИИ-сервисов в UI и
шаблонах промптов. Mind не решает за тебя каким сервисом пользоваться.
Данные передаёшь только ты, в тот сервис которым пользуешься.

### 💾 Локальное хранилище
<img width="400" height="452" alt="Снимок экрана 2026-06-03 003937" src="https://github.com/user-attachments/assets/efebfd10-970f-4ddd-9751-e761fbe2b9af" />

Всё в `%APPDATA%/fresh-mind/`:

- `freshmind.db` — SQLite (записи, теги, сферы, оценки, группы)
- `attachments/` — картинки

Никаких внешних запросов. Бэкап = копия папки в любую синхронизируемую
директорию (Dropbox / OneDrive / Google Drive — на усмотрение пользователя).

Через **Настройки → Бэкап и данные**: экспорт / импорт JSON, полный бэкап
папки, экспорт для ИИ-разбора.

## Сборка из исходников

```bash
npm install
npm run dev     # dev-режим с HMR
npm run dist    # сборка установщиков в release/
```

**Стек:** Electron 35 + electron-vite + React 19 + better-sqlite3 + TipTap.
JavaScript (не TypeScript).
