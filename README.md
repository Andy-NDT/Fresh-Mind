# Fresh Mind

Дневник мыслей и колесо сфер жизни. Часть линейки Fresh.

**Локально. Без облака. Без ИИ внутри.**

Mind не думает за тебя. Не интерпретирует. Не выдаёт советы «попробуй X».
Только отражает, помнит и хранит. А когда захочешь разбора — одной кнопкой
готовит структурированный отчёт для **внешнего** ИИ-сервиса по твоему выбору.

## Что это

Настольное приложение для рефлексии:

- Колесо 20 сфер жизни с шкалой оценок 0-10 на каждый день
- Дневник с тегами, эмодзи настроения, картинками
- Сравнение состояния за разные даты
- «В этот день» — записи из прошлых лет
- Полная локальная работа, никаких облачных сервисов

## Готов к ИИ-аналитике (опционально, снаружи)

Сам Mind ИИ не использует. Но даёт **мост к нему одной кнопкой**: твои
записи и оценки сфер за выбранный период собираются в структурированный
отчёт с готовой инструкцией для разбора. Загружаешь файл в свой ИИ-сервис —
получаешь взгляд со стороны на свои паттерны и динамику. Данные передаёшь
только ты, в тот сервис которым пользуешься.

В окне «Экспорт для ИИ-аналитики»:

- 6 готовых типов разбора (глубокая ретроспектива, анализ одной сферы, помощь
  с решением, поиск слепых пятен, сравнение двух периодов, свой промпт)
- Структурированный отчёт `.txt`: динамика 20 сфер с трендами по третям периода,
  топ-10 ярких записей, паттерны времени и настроений, корреляции (показываются
  только при достаточном объёме данных — чтобы не давать ложных сигналов)
- Инструкция для ИИ зашивается в начало файла — один upload в твой ИИ-сервис,
  и всё готово к разбору. Либо отдельно: кнопка «Копировать промпт» для тех,
  кто предпочитает вставить промпт в чат и приложить файл отдельно

## Где хранятся мои данные

Все данные хранятся локально на твоём компьютере:

```
%APPDATA%/fresh-mind/
```

- `freshmind.db` — база данных SQLite
- `attachments/` — картинки в записях

Делай бэкап через **Настройки → Бэкап и данные**.
